# MentalMuse

A mental health and emotional wellness application focused on anxiety reduction and emotional regulation.

## Features

- Anonymous Community Support
- Voice Journaling
- Breathing Exercises
- Emotional Analysis
- Progress Tracking

## Tech Stack

- React
- TypeScript
- Tailwind CSS
- Vite
- Framer Motion

## Getting Started

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build
